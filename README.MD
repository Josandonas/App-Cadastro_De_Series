# DIO APP Cadastro de Series

> ### Ações do app:
>
>> * Inserir Nova Série
>> * Alterar
>> * Excluir
>> * Listar Series
>> * Visualizar uma única série

### Ambiente

Para funcionar localmente, precisa ter instalado:
* .NET 3.2
* VisualCode 1.55.2

### Build

```
dotnet build
```

### Execução

```
dotnet run
```
